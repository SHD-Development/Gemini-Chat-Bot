# Gemini-Chat-Bot
 An AI chat bot using Google's Gemini API, with prompts and chat history.

# Installation
1. Download the source code.
2. Install [Node.js](https://nodejs.org).
3. Run `npm install` .
4. Edit `config.js` .
5. Customize your prompts at `prompts/` .
6. Run `node .` .
